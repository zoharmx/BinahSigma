SYSTEM ARCHITECTURE
================================================================================

Request Flow:
  1. Client sends request to /v2/analyze with API key
  2. APIKeyAuth validates JWT token and extracts tier
  3. RateLimiter checks tier limits (per-minute/day/month)
  4. LLMOrchestrator sends to primary provider (Mistral)
  5. On provider failure, automatic failover to Gemini/DeepSeek
  6. LLM returns dimensions (clarity, stakeholder, feasibility, ethics)
  7. ScoringEngine calculates index deterministically
  8. QualityValidator checks for generic/vague content
  9. Response includes index, dimensions, metadata, usage info
  10. UsageTracker records request for billing

Industry Weights Example (Healthcare):
  - Clarity: 15%
  - Stakeholder Benefit: 25%
  - Feasibility: 20%
  - Ethical Risk: 40% (highest priority)

Ethical Veto System:
  - Critical Risk: Index capped at 0.40
  - High Risk: Index capped at 0.60
  - Ensures dangerous decisions never score high
